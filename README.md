# README.md

## Getting started
* `npm start`: Start everything via Docker Compose
* `npm test`: Runs Jest + Enzyme tests
* `npm run lint`: Run ESLint on the project

## Architecture
* Typescript 4
* Preact 10
* Webpack
* React Query
* Hooks
* Functional Components
* MongoDB
